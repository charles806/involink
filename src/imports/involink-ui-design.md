# Involink Product UI Design Prompt

Design the full product interface for **Involink**, a mobile-first invoicing platform for Nigerian freelancers.

The product allows users to create invoices, send them to clients, track payments, and send reminders from a simple dashboard.

The interface should feel:

- simple
- fast
- professional
- mobile-first

Users should be able to create and send an invoice in under **3 minutes**.

---

# Design Principles

The UI should prioritize:

Clarity  
Speed  
Minimal learning curve  
Mobile usability

Freelancers should never feel like they are using accounting software.

---

# Recommended UI Style

Modern SaaS minimalism.

Use:

- large white space
- card-based layouts
- bold sans-serif typography
- minimal borders

Typography

Inter  
Poppins

---

# 2026 UI Trends

Include subtle micro-interactions such as:

Glow effect on **Send Reminder** buttons

Card hover animations

Smooth loading states

Use slightly brutalist UI edges to create authenticity and reflect the Nigerian hustle culture.
